# Toy Robot Interview Practice Starter

This workspace contains starter files for four exercises:

1. `01-robot-domain-ts`: Reimplement the toy robot domain in TypeScript.
2. `02-robot-api`: Expose robot domain logic through a REST API (Express).
3. `03-robot-ui`: Build a minimal browser UI to drive the API.
4. `04-code-review-debug`: Review and debug intentionally flawed TypeScript code.

Each folder includes TODOs and minimal scaffolding only.
